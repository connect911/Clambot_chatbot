# Calmbot_chatbot
AI powered chatbot that provide emotional assistance to overcome stress, depression and anxiety

Dependencies that need to install to run server file.
npm install express
npm install openai
npm install path
npm install mysql
npm install axios
npm install child_process
npm install express-session
npm install multer
npm install body-parser

After running the install commands, you should see these dependencies listed in the dependencies section of your package.json file.
